You have an extensive commit history with only a few formatting issues. Your README is polished, but the rest of the code looks rushed and messy. Great work overall!!

Even though, the page is a great start, there is definitely some room for improvement. Here are some of the issues that I saw:

  • HTML is missing the <ol> and <strong> tags.
  • The <span> that starts on line 53 in aboutme.html is being used like a <div>. Spans are for inline styling, not for creating sections for styling. The <span> on line 79 is being used correctly.
  • Why do you have 6 stylesheets? You should have one stylesheet for the entire project, then any stylesheet libraries that you may need (such as Bootstrap). One stylesheet named styles.css would incorporate all the styles from aboutme.css, blog.css, lists.css, and master.css.
  • In blog.html, lines 4-28 are not indented enough.
  • In aboutme.html, lines 4-114 are not indented enough.
  • In aboutme.html, lines 54-57 and 98 are indented too far.
  • In aboutme.html, the <a> tag at the end of line 29 should all be on the same line. Either move the part on line 30 up, or the part on line 29 down.
  • In aboutme.html, the <footer> tag is opened but never closed. See HTML footer Tag for more information on how to properly use a footer tag.
  • In index.html, lines 4-129 are not indented enough.
  • In index.html, the <a> tag at the end of line 30 should all be on the same line. Either move the part on line 31 up, or the part on line 30 down.
  • In index.html, lines 52-55 and 98 are indented too far.
  • In all HTML files, there is far too much whitespace. Having an inconsistent application of whitespace makes your code look messy and unorganized. Examples:
    • In blog.html, there are 13 empty lines.
    • In aboutme.html, there are 41 empty lines.
    • In index.html, there are 63 empty lines.
  • Commit messaging is inconsistent. Examples include but are not limited to: Fixes bootstrap linkpwd and layout errors, adds content. Better commit messages would be fix bootstrap link and add content to homepage and fix layout errors.

Overall, a great effort! Don't be discouraged by the number of comments here. There's quite a bit that's off, but none of it is very difficult to fix.
